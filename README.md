# Tech Solutions – Multi-Page Website

## Live Demo
https://hemanthkumar-00.github.io/synent-task7-techsolutions-hemanthkumar/

## Objective

To design and develop a professional multi-page website for a technology company with a responsive layout and interactive features.

## Steps Performed

1. Created separate pages for Home, About, Services, and Contact sections.
2. Designed a responsive user interface using HTML and CSS.
3. Implemented JavaScript features such as a hamburger menu, contact form validation, and a back-to-top button.
4. Enhanced the website with modern styling and improved user experience.
5. Tested the website across different screen sizes for responsiveness.

## Tools Used

* HTML5
* CSS3
* JavaScript
* VS Code
* Git & GitHub

## Outcome

A professional and responsive multi-page website was successfully developed, demonstrating frontend development skills and providing an engaging user experience.
